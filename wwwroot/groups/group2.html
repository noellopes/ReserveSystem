Groupo 2 Objetivos do Trabalho

Group Objective

Este grupo necessita de criar um programa de cálculo automático para os preços de quartos de hotel, tendo em conta:

  • Eventos locais
  • Preço base do quarto
  • Sazonalidade e altura do ano
  • Serviços de quarto

Sendo necessário adicionar um API de meteorologia para associar a um programa que pode aumentar ou diminuir os preços do quarto consuante a sazonalidade, tem de existir uma ligação as tabelas do grupo 3, serviços de quarto adicionais, que possam influenciar o preço final do quarto, é necessário um utilizador privilegiado que tem a função de inserir, alterar e eliminar a tabela de preços de quartos e acrescentar descontos ou inflações de preços consuante eventos locais, sazonalidade e serviço de quarto, para criar este projeto é necessário ainda criar todas as bases de dados associadas.
